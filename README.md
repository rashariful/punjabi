# Project Overview

This project is a modern **online clothing store** designed for selling **Punjabi collections** such as traditional and modern designs.
It provides an easy shopping experience with responsive UI and optimized performance.

The system is built using a **modern frontend stack with reusable components**, ensuring scalability and maintainability.

# Installation

Make sure **Node.js** and **npm** are installed.

### Clone Repository

```bash
git clone https://github.com/rashariful/punjabi
```

### Navigate to Project Folder

```bash
cd punjabi
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The development server will start with **hot reload enabled**.
